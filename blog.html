Question-1: What are the different ways to select an element in the DOM?

In JavaScript there are many ways to select element in dom. We can select by id, classname, tag name, query selectors like css selectors etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML catches the html inside the element its captured and innerText captures the texts only inside the dom element, textContent is used to get or set content of an element.

Question-3: What is event delegation in the DOM?

By this we can attatch one eventlistener to a parent node by that we can get all individual element events inside that parents node. by doing that we dont have to add eventlistener for individual elements.

Question-4: What is event bubbling in the DOM?

It is a mechanism in DOM that describe how events propagate through the dom tree.

Question-5: How do you create, add, and remove elements using JavaScript?

To create new element we can use createElement() method, to add element we can use appendChild(),insertBefore(),innerHTML method, To remove we can use removeChild(), remove() method.
